# notesman

util to help manage my todos in my notes

> cargo install --path .

## Use

> notesman --help
notesman 0.5.4

USAGE:
    notesman <todo_file>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

ARGS:
    <todo_file>    The source todo file to be processed.
